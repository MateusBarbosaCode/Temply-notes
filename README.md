 Temply - Minimalist & Elegant Notes
O Temply é um bloco de notas progressivo (PWA) focado em minimalismo, elegância e performance. Desenvolvido para oferecer uma experiência de escrita fluida e livre de distrações, combinando um design premium com tecnologias modernas de web app.

Status: 🚀 Em produção / Funcional

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

Frontend: HTML5 semântico e CSS3 (Custom Properties para temas).

Mobile First: Design responsivo adaptado para qualquer tamanho de tela.

PWA (Progressive Web App): * manifest.json para instalação nativa no Android, iOS e Windows.

Suporte a ícones de alta resolução e Splash Screen personalizada.

Experiência de "App nativo" rodando diretamente no navegador.

Persistence: LocalStorage para salvamento automático das notas (Offline First).

💎 Diferenciais Técnicos
Instalabilidade: Graças à configuração do Web Manifest, o Temply pode ser instalado na tela inicial do celular com o ícone dourado exclusivo.

Design Premium: Interface baseada em tons de preto e dourado, focada no mercado de luxo e alta produtividade.

Código Limpo: Estrutura de arquivos organizada seguindo princípios de Engenharia de Software para fácil manutenção.

🚀 Como instalar
Acesse o link do projeto pelo seu navegador (Chrome ou Safari).

No Android/PC: Clique no ícone de "Instalar" na barra de endereços.

No iOS: Vá em "Compartilhar" > "Adicionar à Tela de Início".
